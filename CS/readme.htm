Developer Express Example
How to implement a custom inplace editor for appointments

This example demonstrates how you can implement a custom editing form that will be invoked instead of the default inplace editor. See the How to: Customize an Inplace Editor (http://documentation.devexpress.com/#WindowsForms/CustomDocument2301) document for a step-by-step guide.
